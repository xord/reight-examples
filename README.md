# Reight Examples

レトロゲームエンジン Reight (R8) のサンプルコード集です。

## 使い方

```sh
$ git clone https://github.com/xord/reight-examples.git
$ cd reight-examples
$ bundle install
$ bundle exec r8 hello
```
